Meus serviços

Front End

Front end engloba todo o espaço visível pelo usuário, assim como a página que você está vendo. Propõe apresentar uma experiência agradável ao usuário, tanto visualmente quanto na interatividade. Aqui utilizo tecnologias como JavaScript e ReactJs.

Back End

Back end representa o que acontece fora do campo de visão do usuário, como a integração com banco de dados, o backend é a estrutura que possibilita a operação do sistema. Aqui utilizo tecnologias como NodeJS e Java.

Banco de Dados

Um banco de dados é uma coleção organizada de informações - ou dados - estruturadas, normalmente armazenadas eletronicamente em um sistema de computador. Aqui utilizo o sistema de gerenciamento MongoDB
