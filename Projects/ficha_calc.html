    Descrição Técnica do Projeto "Calc.js"

    O projeto "Calc.js" é uma calculadora simples desenvolvida utilizando HTML, CSS e JavaScript. Possui as funcionalidades básicas de uma calculadora, como adição, subtração, multiplicação, divisão e operações com números decimais.
    Repositório no GitHub: https://github.com/raulaugusto/Calc.js
    Link para o projeto: https://raulaugusto.github.io/Calc.js/

    Tecnologias Utilizadas:


    • HTML/CSS/JavaScript: As tecnologias fundamentais para o desenvolvimento web são empregadas para criar uma interface de usuário responsiva e amigável.

    Funcionalidades Principais:


    • Operações Básicas: Adição, Subtração, Multiplicação, Divisão.
    • Operações de Ponto Flutuante: Suporte para números decimais.
    • Limpar e Apagar: Funcionalidade para limpar o visor da calculadora e apagar o último caractere.

    Visão Geral da Arquitetura:


    • Arquitetura Cliente-Servidor: O projeto segue uma arquitetura cliente-servidor, onde o cliente (navegador do usuário) solicita recursos e interage com a aplicação web, enquanto o servidor (GitHub Pages) fornece os arquivos estáticos da aplicação.
